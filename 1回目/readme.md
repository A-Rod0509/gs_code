①課題内容（どんな作品か）-必要あれば操作方法等こちらに記載
内容はジャンケンとRPGを作成しました。勇者となってモンスターを倒します。
１．コマンドを押してジャンケンの手を選ぶ
２．勝てば相手にダメージ、あいこは０、負けはこちらにを０にさせれば勝ちです。
３．HPが０になるとゲームオーバーでポップアップを表示
