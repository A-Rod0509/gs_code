①課題内容（どんな作品か）-必要あれば操作方法等こちらに記載
内容はチーズアカデミーサイトの作成です。スクロールすることで、
各セクションを浮き上がらせて表示します。また、チーズアカデミーを
卒業生として、ストリートファイターのリュウを登場させました。
操作（左右移動のみ）可能です。

②工夫した点・こだわった点
見やすさ、見ていただく方に魅力的なサイトに
思ってもらえるよう各ページ表示時に下記アニメーションを設定しました。
・スクロールさせることで各セクションを浮き上がらせるように表示
・自動再生可能なスライドショー
・おしゃれなSubmitボタン
また、面白みを出そうと思い、ストリートファイターのリュウを
を登場させてます。

③質問・疑問（あれば）
アニメーションの設定方法のバリエーションを増やしたい。
(本課題では下から浮き上がらせるアニメーションばかりなため)
slickライブラリを活用し、スライドショーを設定しましたが、
表示がきちんとできていないです。

④その他（感想、シェアしたいことなんでも）
反省点としては、フォントサイズが小さく同じ書体のためメリハリがないこと、
アニメーションの設定でハマってしまい、レスポンシブデザインを
意識した作りこみができなかったり、アニメーションの動きが小さかったり、
ストリートファイターが未完成だった点。
皆さんの作品を拝見し、インスピレーションを受けました。
木曜の勉強会等でどのようにして発見、実現したかをヒアリングして
自分のスキルアップに努めたいと思います。
